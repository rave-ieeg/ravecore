# 'RAVE' constants

Constant variables

## Usage

``` r
LOCATION_TYPES

SIGNAL_TYPES

IMPORT_FORMATS

YAEL_IMAGE_TYPES

MNI305_to_MNI152
```

## Format

An object of class `character` of length 5.

An object of class `character` of length 5.

An object of class `list` of length 7.

An object of class `character` of length 10.

An object of class `matrix` (inherits from `array`) with 4 rows and 4
columns.
